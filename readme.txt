Easily add and display QR code directly from Advanced Custom Fields

== Description ==

Create field, generate QR on fly and display it using shortcode of Advanced Custom field.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-qrcode` folder into your `wp-content/plugins` folder
2. Activate the ACF QR Code plugin via the plugins admin page
3. Create a new field via ACF and select the QR code type
4. Read the description above for usage instructions
